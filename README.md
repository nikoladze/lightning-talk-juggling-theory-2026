# Lightning talk Juggling Theory

I've created this presentation for a 10 minute lightning talk on juggling theory at a socializing event at my employer ([inovex](https://github.com/inovex)).

Besides introducing my colleagues to the cool world of juggling theory, this was also an exercise for me in AI assisted coding. The juggling simulator program and most of the template setup has been done with help by claude code.

The presentation is deployed at

- https://nikoladze.github.io/lightning-talk-juggling-theory-2026 (german version)
- https://nikoladze.github.io/lightning-talk-juggling-theory-2026/en (english version)

To just play with the juggling simulator, go to

- https://nikoladze.github.io/lightning-talk-juggling-theory-2026/juggling.html

Enjoy!

# References on Juggling Theory

Did you like it? Here are some of the references i used and other fun stuff i recommend:

- https://en.wikipedia.org/wiki/Siteswap
- https://www.youtube.com/watch?v=38rf9FLhl-8 (mathematics of juggling lecture by Allen Knutson)
- https://www.youtube.com/watch?v=ELvedTUcjPo (a TEDx talk)
- https://www.qedcat.com/articles/juggling_survey.pdf (survey on mathematical juggling by Burkard Polster, he also wrote [a whole book](https://link.springer.com/book/10.1007/b98883))
- https://arxiv.org/pdf/math/0302257 (paper on the probability of juggling patterns when "juggling randomly")

# Juggling simulators

Some great (proper) juggling simulator programs:

- https://jugglinglab.org (well maintained, lots of features, Android app available)
- https://jongl.de (not maintained anymore, but great 3D graphics and lots of extras, e.g. bouncing balls, chain saws, you name it)
- https://passist.org (runs in browser, 3D graphics, including passing patterns)
- https://ydgunz.github.io/gunswap (runs in browser, 3D graphics, supports hand movements)
